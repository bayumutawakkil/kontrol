STRUKTUR FILE PROYEK
═══════════════════════════════════════════════════════════════════

FILE YANG SELESAI (7 File Java + 1 SQL + 4 Dokumentasi):

1. database_kas.sql
   - Skema database MySQL untuk Kas Organisasi
   - Tabel: kas_organisasi
   - Pernyataan CREATE DATABASE dan CREATE TABLE
   - Penyisipan data contoh (Iuran, Sponsor, Proker, Konsumsi)
   - Lokasi: d:\Project\kontrol\database_kas.sql

2. KoneksiDB.java
   - Manajemen koneksi JDBC
   - Konfigurasi database (URL, USER, PASSWORD)
   - Metode testConnection() untuk validasi koneksi
   - Lokasi: d:\Project\kontrol\src\KoneksiDB.java
   
3. Transaksi.java (Kelas Induk Abstrak)
   - Kelas super abstrak untuk semua transaksi
   - Atribut: id, nominal, keterangan, tanggal
   - Metode abstrak: getTipe(), getKategori()
   - Mendemonstrasikan: Inheritance, Polymorphism, Encapsulation
   - Formatting: getNominalFormatted(), getTanggalFormatted()
   - Lokasi: d:\Project\kontrol\src\Transaksi.java

4. Pemasukan.java (Subclass)
   - Extends Transaksi (Inheritance)
   - Field khusus: sumberDana (Iuran/Sponsor/Donasi)
   - Override metode: getTipe() = "PEMASUKAN"
   - Override metode: getKategori() = sumberDana
   - Lokasi: d:\Project\kontrol\src\Pemasukan.java

5. Pengeluaran.java (Subclass)
   - Extends Transaksi (Inheritance)
   - Field khusus: keperluan (Proker/Konsumsi/Perlengkapan)
   - Override metode: getTipe() = "PENGELUARAN"
   - Override metode: getKategori() = keperluan
   - Lokasi: d:\Project\kontrol\src\Pengeluaran.java

6. KasDAO.java (Interface)
   - Mendefinisikan kontrak operasi CRUD
   - Metode: tambahTransaksi, getAllTransaksi, getTransaksiById,
             hapusTransaksi, getTotalPemasukan,
             getTotalPengeluaran, getSisaKas
   - Lokasi: d:\Project\kontrol\src\KasDAO.java

7. ManajemenKas.java (Implementasi DAO)
   - Implements interface KasDAO
   - Operasi CRUD lengkap dengan JDBC
   - Menggunakan ArrayList<Transaksi> (Collection Framework)
   - Penggunaan PreparedStatement dan ResultSet
   - Penanganan exception untuk SQLException
   - Kalkulasi matematis (Total Pemasukan, Pengeluaran, Sisa Kas)
   - Lokasi: d:\Project\kontrol\src\ManajemenKas.java

8. MainApp.java (Aplikasi CLI)
   - Entry point aplikasi utama
   - Menu CLI interaktif dengan do-while loop
   - Switch-case untuk logika menu
   - Penanganan exception untuk InputMismatchException
   - Fitur: Tambah Pemasukan/Pengeluaran, Lihat Riwayat, Hapus, Laporan
   - Lokasi: d:\Project\kontrol\src\MainApp.java

9. README.md
   - Dokumentasi proyek lengkap
   - Tabel cakupan kriteria penilaian UAS
   - Instruksi setup database dan aplikasi
   - Panduan penggunaan sistem kas organisasi

   - Lokasi: d:\Project\kontrol\README.md

10. SETUP_GUIDE.md

11. PROJECT_SUMMARY.txt

FITUR UTAMA APLIKASI KAS ORGANISASI
═══════════════════════════════════════════════════════════════════

1. Tambah Pemasukan (Iuran Anggota/Sponsor/Donasi)
   - Input sumber dana
   - Input nominal, keterangan, dan tanggal
   - Validasi input dengan exception handling

2. Tambah Pengeluaran (Program Kerja/Konsumsi/Perlengkapan)
   - Input keperluan
   - Input nominal, keterangan, dan tanggal
   - Validasi input dengan exception handling

3. Lihat Riwayat Transaksi
   - Menampilkan semua transaksi (diurutkan berdasarkan tanggal)
   - Format tabel yang rapi dengan ID, Tanggal, Tipe, Kategori, Nominal
   - Menggunakan Collection Framework (ArrayList)

4. Hapus Transaksi
   - Hapus transaksi berdasarkan ID
   - Konfirmasi sebelum menghapus
   - Validasi ID dengan exception handling

5. Laporan Keuangan & Sisa Kas
   - Total Pemasukan (sum dari semua PEMASUKAN)
   - Total Pengeluaran (sum dari semua PENGELUARAN)
   - Sisa Kas (Total Pemasukan - Total Pengeluaran)
   - Status kesehatan keuangan organisasi
   - Format Rupiah (Rp) yang sesuai

SKEMA DATABASE
═══════════════════════════════════════════════════════════════════

Database: kas_organisasi_db
Tabel: kas_organisasi

Kolom:
- id (INT, PRIMARY KEY, AUTO_INCREMENT)
- tipe (VARCHAR(20), NOT NULL) → 'PEMASUKAN' atau 'PENGELUARAN'
- nominal (DOUBLE, NOT NULL)
- keterangan (VARCHAR(255), NOT NULL)
- sumber_dana (VARCHAR(100)) → Untuk PEMASUKAN (Iuran/Sponsor/Donasi)
- keperluan (VARCHAR(100)) → Untuk PENGELUARAN (Proker/Konsumsi/dll)
- tanggal (DATE, NOT NULL)
- created_at (TIMESTAMP, DEFAULT CURRENT_TIMESTAMP)

CARA MENJALANKAN APLIKASI
═══════════════════════════════════════════════════════════════════

1. Setup MySQL Database:
   mysql -u root -p < database_kas.sql

2. Konfigurasi KoneksiDB.java:
   - Set username MySQL (default: root)
   - Set password MySQL (default: kosong)

3. Tambahkan MySQL JDBC Driver ke folder lib/
   - Download mysql-connector-java-8.0.33.jar
   - Letakkan di d:\Project\kontrol\lib\

4. Compile (Windows):
   javac -d bin -cp "lib\mysql-connector-java-8.0.33.jar" src\*.java

5. Run (Windows):
   java -cp "bin;lib\mysql-connector-java-8.0.33.jar" MainApp
